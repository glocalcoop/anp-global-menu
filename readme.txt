=== Activist Network Global Menu ===
Contributors: Pea, Glocal
Tags: navigation, multisite, global menu
License: GPL3

A Wordpress plugin that adds a global menu to all sub-sites in a multi-site network.

==== Usage ====
- Network active the plugin
- Select a menu to use as the global menu in Network Settings > Activist Network Global Menu

The menu will appear on sub-sites only.

==== Known Issues ====
- Part of the menu appears underneath the left column in the default twenty-fifteen them.
